Ugly Number Checker

What is an Ugly Number?

In mathematics, Ugly Numbers are positive integers whose prime factors are limited to 2, 3, and 5. By convention, 1 is typically included as an ugly number.

How to Justify?

To determine if a number is "ugly," divide it repeatedly by 2, 3, and 5 as long as it remains divisible. If the final remaining value is 1, the number is ugly. If any other prime factor remains, it is not.

How to Use This App

1. Enter any positive integer into the input field.
2. Click the "Check" button.
3. View the result below the button to see if your number fits the criteria.
